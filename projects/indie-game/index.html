Indie Game Title

Completed: December 2023 Category: Game Development Platform: PC, Mac, Linux

This 2D platformer game was developed in Unity over the course of six months. It features hand-drawn artwork, original music, and innovative gameplay mechanics that challenge traditional platforming conventions.

The game follows the journey of a character who discovers they can manipulate the flow of time, allowing for unique puzzle-solving and combat scenarios. With over 30 levels spread across 5 distinct worlds, the game offers approximately 8-10 hours of gameplay.

Features

  • Innovative time-manipulation mechanics
  • Hand-drawn art style with fluid animations
  • Original soundtrack composed specifically for the game
  • Progressive difficulty curve with balanced challenges
  • Hidden collectibles and secret areas
  • Narrative-driven gameplay with voiced cutscenes

Gallery

Technical Details

The game was developed using Unity 2022.2, with C# for all gameplay programming. Custom shaders were written in HLSL to achieve the unique visual effects, particularly for the time-manipulation mechanics. The game features a modular architecture with a custom state machine for character control, allowing for easy expansion of abilities and movement options.

All artwork was created using Aseprite for pixel art and animations, while audio was composed using FL Studio and integrated with FMOD for dynamic audio transitions based on gameplay state.
